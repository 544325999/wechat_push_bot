## 
📦 一个基于 PHP webman框架开发的消息推送.  
1.抖音同款推送 php实现服务端.  
2.微信公众号服务端接入米家io 可通过公众号发送命令调用米家生态.
## 环境要求
- PHP >= 7.2
- [Composer](https://getcomposer.org/) >= 1.0
## Installation
```shell
编辑 config/wechat.php 配置
定时任务执行时间配置 process/Task.php
```
```shell
composer install
```
```shell
php start.php start
```
```shell
米家相关配置信息
cp .env.example .env
```
